## Getting Started

Puedes correr el repositorio localmente siguiendo los siguientes pasos


### Installation

1. Clone the repo

```sh
git clone https://github.com/niicodeer/real-estate-website.git
```

3. Install NPM packages

```sh
npm install
```

4. Run the project

```sh
npm run dev
```
